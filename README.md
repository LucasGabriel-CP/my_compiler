# my_compiler

Compiler made from scratch to a tiny language for Compiler Design classes

Base language: `c++`

Language type: `LL(1)`

Phases complited:

* Lexical Analysis
    * Used a finite automaton
* Syntax Analysis
    * Used tabular preditive analysis
    * Created a Syntax Tree
* Semantic Analysis
    * Used the Syntax Tree from previous step
    
Future phases:

* Code Optmiser
* Code Generation
